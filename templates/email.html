Для Ubuntu необходимо выполнить следующие действия:
  • Устанавливаем следующие пакеты:
  • sudo apt -y install wireguard openresolv
  • После утстановки пакетов,
    необходимо положить конфиг файл wg-{{ user_name }}.conf
    в папку /etc/wireguard/
  • sudo cp ~/Downloads/wg-{{ user_name }}.conf /etc/wireguard/
  • если у Вас русская локаль, то:
  • sudo cp ~/Загрузки/wg-{{ user_name }}.conf /etc/wireguard/
  • и выполнить ледующие команды:
  • sudo systemctl start wg-quick@wg-{{ user_name }}.service
  • sudo systemctl enable wg-quick@wg-{{ user_name }}.service
